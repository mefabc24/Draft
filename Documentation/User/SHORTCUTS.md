# Keyboard Shortcuts

These are Draft's default keyboard shortcuts. They can be customized and reset under **Settings > Shortcuts**.

## General

| Default shortcut | Action |
| --- | --- |
| `Ctrl + S` | Save the current file |
| `Ctrl + O` | Open a file |
| `Ctrl + F` | Toggle Find and Replace |
| `Ctrl + Z` | Undo |
| `Ctrl + Shift + Z` | Redo |
| `Ctrl + D` | Duplicate the current line |
| `Ctrl + Alt + U` | Toggle current-line capitalization |
| `Ctrl + Shift + U` | Convert the selection to uppercase |
| `Ctrl + Shift + L` | Convert the selection to lowercase |
| `Ctrl + Shift + Up` | Move the current line or selected lines up |
| `Ctrl + Shift + Down` | Move the current line or selected lines down |
| `Ctrl + Left` | Move the cursor one word left |
| `Ctrl + Right` | Move the cursor one word right |
| **Not set** | Move the cursor to the start of the next line |
| `Ctrl + Shift + Left` | Extend the selection one word left |
| `Ctrl + Shift + Right` | Extend the selection one word right |
| `Ctrl + Shift + Alt` + left drag or double-click | Add a separate selection |
| `Enter` | Continue the current Markdown list or quote |
| `Tab` | Indent an empty Markdown list item |

The shortcut for moving to the start of the next line is intentionally unassigned. When configured, it performs the equivalent of `Down` followed by `Home`.

## Floating Markdown Toolbar (FMT)

| Default shortcut | Action |
| --- | --- |
| `Ctrl + B` | Bold |
| `Ctrl + I` | Italic |
| `Ctrl + U` | Underline |
| `Ctrl + E` | Inline code |
| `Ctrl + Shift + S` | Spoiler |
| `Ctrl + Shift + H` | Highlight |
| `Ctrl + /` | HTML comment |
| `Ctrl + Shift + X` | Strikethrough |
| `Ctrl + K` | Link |
| `Ctrl + Alt + I` | Image |
| `Ctrl + N` | Normal text |
| `Ctrl + 1` through `Ctrl + 6` | Heading 1 through Heading 6 |
| `Ctrl + Shift + E` | Edit the Markdown behind selected preview content |
| `Enter` | Confirm an edit in a toolbar menu |
| `Esc` | Close the toolbar or cancel an open edit menu |

## Quick Insert Menu (QIM)

| Default shortcut | Action |
| --- | --- |
| `Ctrl + Space` | Open the Quick Insert Menu at the cursor position |
| `Shift` + left click | Insert an item and keep the menu open |

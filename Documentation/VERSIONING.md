# Versioning

Update the version in the following locations before creating a new release.

## Desktop App

File:

`Draft.Wpf/Draft.csproj`

Properties:

```xml
<Version>...</Version>
<AssemblyVersion>...</AssemblyVersion>
<FileVersion>...</FileVersion>
```

## Web App

File:

`Draft.Web/package.json`

Property:

```json
"version": "..."
```

File:

`Draft.Web/package-lock.json`

Properties:

```json
"version": "..."
```

The lock file contains the root package version near the top of the file and inside the root package entry. These values can be updated manually or regenerated by running `npm install` in `Draft.Web`.

## Generated Release Files

The following files are generated as part of the release process:

```text
Releases/releases.win.json
Releases/assets.win.json
```

Do not update these files manually when changing the app version. Regenerate them through the release process instead.
